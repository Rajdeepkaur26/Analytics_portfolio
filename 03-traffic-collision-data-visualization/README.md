## Project Overview
This project analyzes **traffic collision data from Montgomery County, Maryland**, using visual analytics to uncover patterns, hotspots, and risk factors. The goal was to support **data-driven decision-making** for public safety and policy interventions.

---

## Business / Policy Problem
Traffic collisions pose serious risks to public safety and infrastructure.
Raw datasets alone do not easily reveal:
- High-risk locations
- Time-based collision patterns
- Contributing factors

This project demonstrates how **business intelligence and visualization** can translate complex data into actionable insights.
